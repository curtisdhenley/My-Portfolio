# My Portfolio

Welcome to my portfolio! This landing page provides an introduction to who I am and showcases my skills, projects, challenges, blog posts, and how to get in touch.

## Sections

1. **About**: Get to know more about me and my background.
2. **Skills**: Explore the skills I've developed throughout my journey.
3. **Projects**: Highlights of my top 3 full-stack applications.
4. **Challenges**: Demonstrates my understanding of programming fundamentals.
5. **Blog**: Features the four most recent blog posts pulled from my blog API.
6. **Contact**: Reach out to me! Utilize the form to send a message.

## Tech Stack

- **JavaScript**: Core programming language for interactivity.
- **Bootstrap**: CSS framework for responsive and visually appealing design.
- **HTML**: Markup language for structuring the web page.
- **CSS**: Stylesheet language for additional styling.

## How to Use

1. **Clone the Repository**: `git clone https://github.com/your-username/My-Portfolio.git`
2. **Open the HTML File**: Open the `index.html` file in your preferred web browser.
3. **Navigate the Sections**: Use the navigation links to explore different sections of the portfolio.
